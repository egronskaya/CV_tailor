# CV Tailoring Instructions

# Matching Instructions
1. Extract key requirements and skills from job posting
2. Identify matching experiences from base CV
3. Use similar terminology as job posting
4. Quantify achievements where possible
5. Highlight relevant projects and responsibilities
6. Order experiences by relevance to position
7. Adjust technical skills section to prioritize required skills
8. Customize professional summary to match job focus

# Optimization Rules
1. Use industry-standard keywords from posting
2. Include exact skill names as mentioned in posting
3. Match seniority level terminology
4. Reflect company values in achievements
5. Incorporate domain-specific language
6. Maintain authenticity while aligning with requirements
7. Ensure all critical requirements are addressed
8. Preserve chronological order within sections

# ATS Optimization
1. Use standard section headings
2. Avoid tables and columns
3. Use common fonts
4. Include keywords in context
5. Spell out acronyms
6. Use standard bullet points
7. Keep formatting simple
8. Include location and job titles exactly as posted

# Content Guidelines
1. Focus on relevant achievements
2. Remove unrelated experiences
3. Expand on matching qualifications
4. Adjust technical depth based on role
5. Highlight transferable skills
6. Include relevant certifications
7. Add role-specific metrics
8. Customize project descriptions

# Quality Checks
1. Verify all requirements are addressed
2. Check keyword density
3. Confirm readability
4. Validate formatting
5. Review date consistency
6. Check contact information
7. Verify links and references
8. Proofread thoroughly
